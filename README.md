# UI Skills

Reusable Codex skills for UI work.

## tma-ui

`tma-ui` helps build and review polished Telegram Mini App interfaces. It packages reusable guidance from `IlyaGrshin/wallet_animations`: Telegram WebView-safe routing, native TWA surfaces, safe-area layouts, Apple/Material platform skins, Telegram theme tokens, and performant motion patterns for tabs, modals, popovers, page transitions, and wallet-like flows.
