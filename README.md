# UI Skills

Reusable Codex skills for UI work.

## tma-ui

`tma-ui` helps build and review polished Telegram Mini App interfaces. It bundles the `IlyaGrshin/wallet_animations` source so agents can reuse its React components directly, then adapt Telegram WebView-safe routing, native TWA surfaces, safe-area layouts, Apple/Material skins, theme tokens, and motion patterns to other projects.
