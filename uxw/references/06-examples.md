# UXW Examples

## Buttons

| Weak | Better | Why |
|---|---|---|
| Хочу удалить | Удалить | Button is the user's action. |
| Сохраняю | Сохранить | Avoid system present tense. |
| Я понял | Понятно | Standard acknowledgement. |
| ОК | Понятно | Clearer in Russian UI. |

## Forms

| Weak | Better | Why |
|---|---|---|
| Введите номер телефона | Номер телефона | Label names the data. |
| Напишите комментарий | Комментарий | Avoid instructing obvious typing. |
| Начните писать | Поиск | Name the field or action. |
| Отчество | Отчество, если есть | Optionality belongs in visible copy. |

## Errors

| Weak | Better | Why |
|---|---|---|
| Ой! Что-то пошло не так. Извините нас, пожалуйста. | Не загрузилось. Попробуйте снова | Calm, actionable. |
| К сожалению, не удалось загрузить данные | Не получилось загрузить данные | Removes apology-like framing. |
| Произошла ошибка авторизации | Войдите снова | User action first. |

## Links

| Weak | Better | Why |
|---|---|---|
| Подробнее здесь | Подробнее о тарифе | Names the destination. |
| Нажмите тут | Открыть условия | Names the action. |
